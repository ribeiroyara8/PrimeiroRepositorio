# Meu primeiro repositorio 
Este é o meu primeiro projeto que enviei para o Github.

## Como enviar projetos para o Github
- Precisa ter o **GIT** instalado e o **Github Desktop** 
- Abrir o **CMD** na pasta do projeto e executar o comando : `git init`
- Fazer o **commit** no **Github Desktop** e depois clicar em `push` para enviar
    - Se é a primeira vez, precisa clicar em `publish`.

    - Lembrar de, preferencialmente, deixar o projeto **public**

## Como baixar projetos
Abrir o **CMD** e digitar `git clone [link do repositorio]`
